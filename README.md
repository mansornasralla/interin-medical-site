# M-Nexus Medical - فريق انترين الطبي

## Local
```cmd
npm start
```

## Railway Variables
Add these variables in Railway:
```text
SUPABASE_URL=https://YOUR_PROJECT_REF.supabase.co
SUPABASE_SERVICE_KEY=YOUR_SUPABASE_SECRET_OR_SERVICE_KEY
```

## تشغيل الموقع على جهازك
1. فك الضغط عن الملف.
2. افتح Terminal / CMD داخل المجلد.
3. نفذ: `npm install` مرة واحدة إذا كانت الحزم غير مثبتة.
4. نفذ: `npm start`
5. افتح الرابط الذي يظهر، غالباً: `http://localhost:3000`
